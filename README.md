# Arabic Quotes VS Code Extension

This extension displays an Arabic wisdom quote (حكمة) in the VS Code status bar.
Every 10 seconds the quote changes automatically with a new color and smooth animation.
You can also click the status bar to show the full quote in a popup.

## Features
- The extension load:
  - `quotes`
  - `azkar`
  - `ayat`
- Automatic color change every cycle
- Works on left side of status bar
- Optional popup on click
- Smooth animated floating bubble


## How to Use
Install the extension (VSIX), and it will start automatically.
Setting available:

| Setting | Description |
|--------|-------------|
| `Arabic Quotes` | Choose between `quotes`, `adhkar`, or `ayat` |

| Action                     | Result        |
| -------------------------- | ------------- |
| Close bubble tab           | Only hides it |
| Run `Hukm: Open Bubble`    | Restores it   |
| Click the status bar quote | Reopens it    |
| Add keyboard shortcut      | Quick access  |



Press Ctrl + Shift + P

Type: Hukm: Open Bubble

Press Enter

This will reopen the floating quotes window.
